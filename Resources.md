# 📚 Frontend Bootcamp Resources

This file contains helpful links, tools, and references to support your learning journey through **Frontend Bootcamp 2.0**. All resources are curated to match the topics covered in the curriculum and promote best practices.

---

## 🌐 General Web Dev Tools & Documentation

- [MDN Web Docs](https://developer.mozilla.org) – Official documentation for HTML, CSS, and JavaScript.
- [W3Schools](https://www.w3schools.com) – Beginner-friendly tutorials and interactive examples.

---

## 🧰 Accessibility Resources

- [WAI-ARIA Authoring Practices](https://www.w3.org/WAI/ARIA/apg/) – Official guide to accessible web component patterns.
- [WebAIM](https://webaim.org/resources/designers/) – Practical accessibility resources for developers and designers.

---

## 🎯 CSS Guides & Resources

- [CSS-Tricks](https://css-tricks.com) – Articles, guides, snippets, and layout techniques for modern CSS.

### CSS Resets

- [Eric Meyer’s Reset CSS](https://meyerweb.com/eric/tools/css/reset/) – Classic stylesheet to remove browser inconsistencies.
- [Josh W. Comeau’s Modern CSS Reset](https://www.joshwcomeau.com/css/custom-css-reset/) – Modern reset for clean base styles.
- [Normalize.css](https://necolas.github.io/normalize.css/) – Preserves useful defaults and improves cross-browser rendering.

---

## 🛠 Tools & Playground Platforms

- [CodeSandbox](https://codesandbox.io) – Online editor for frontend projects with support for folder structure.
- [CodePen](https://codepen.io) – Live HTML/CSS/JS code snippets and quick prototyping.
- [JSFiddle](https://jsfiddle.net) – Playground for HTML/CSS/JS experiments and sharing.

---

## 🧠 Bonus Learning Platforms

- [Frontend Mentor](https://www.frontendmentor.io) – Real-world frontend practice challenges.
- [freeCodeCamp](https://www.freecodecamp.org) – Hands-on coding lessons and certifications.
- [CSS Battle](https://cssbattle.dev) – Fun CSS-based visual challenge platform.
